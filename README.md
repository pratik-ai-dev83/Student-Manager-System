# STUDENT MANGAER SYSTEM 
 A simple and effcient Student Managemant System built using python.
 this project allows you to manage student records like adding, viewing, searching,
 and deleting student.


# FEATURES
- Add new student
- View all students
- Search student by name
- Delete student
- Data stored in file

# TECHNOLOGIES USED
- Python
- File Handling
- OOP

# HOW TO RUN?
1. Clone the repository:
   git clone https://github.com/pratik-ai-dev83/Student-manager-System.git
2. Go to project folder:
   cd Student-Manager-System
3. Run the program:
   python main.py

# PROJECT STRUCTURE
Student-Manager-system/
|-- main.pyc         # Main program
|-- student.py       # Student class 
|-- storage.py       # File handling
